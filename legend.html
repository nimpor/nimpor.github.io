    Standard Befehle

    Befehle die jeder von Start aus benutzen kann.

    Befehl Level Beschreibung
    /accept [Spielername] 0 Akzeptiert die Teleportanfrage des Spielers
    /animation 0 Deaktiviert/Aktiviert Animationen für dich
    /cc 0 Cleart deinen Chat
    /hide 0 Macht das Scoreboard an der Seite unsichtbar
    /ping 0 Deaktiviert/Aktiviert die Markierung falls dein Name im Chat geschrieben werden sollte
    /show 0 Macht das Scoreboard an der Seite sichtbar
    /verify 0 Verifiziert deinen Minecraft Account mit dem Account der Website

    Battlepass

    Folgende Befehle schaltest du mit dem Battlepass ab Level X frei

    Befehl Level Beschreibung
    /spawn 3 Teleportiert dich zum Spawn
    /homes 8 Listet alle deine Homes auf
    /sethome [Name] 8 Setzt ein Home an deiner aktuellen Location mit dem angegebenen Namen
    /home [Name] 8 Teleportiert dich zu dem angegebenen Home
    /delhome [Name] 8 Löscht das angegebene Home
    /tpa [Spielername] 10 Schickt eine Teleportanfrage zu dem angegebenen Spieler
    /wb 24 Ruft eine mobile Workbench auf deiner aktuelle Location auf
    /top 29 Teleportiert dich nach oben
    /stonecutter 34 Ruft einen mobilen Stonecutter an deiner aktuellen Location auf
    /anvil 49 Ruft einen mobilen Anvil an deiner aktuellen Location auf
    /ec 54 Ruft deine Enderchest auf

    Base

    Alle Befehle bezogen auf das Base System

    Befehl Level Beschreibung
    /base create [Basisname] 0 Erstellt eine default Base an deiner aktuellen Location mit dem angegebenen Basisnamen
    /base accept [Basisname] 0 Nimmt eine Einladung der Basis an
    /base leave [Basisname] 0 Verlässt die angegebene Basis
    /base info (Basisname) 0 Erhalte eine Info über die Basis - der Basisname Parameter ist hierbei Optional
    /base list 0 Erhalte eine Liste in welchen Basen du dich befindest
    /base circle [Radius] 0 Passt deine bereits erstellte Basis auf einen Kreis mit folgenden Radius an
    /base rectangle [X,Z] [X,Z] 0 Passt deine bereits erstellte Basis auf ein Rectangle mit angegebenen Koordinaten an. Beispiel: /base rectangle 0,20 20,0
    /base color [Farbe als sechsstelliger Hex Code] 0 Ändert die Farbe der Basis. Farbe ist hierbei eine Hex-Color
    /base invite [Spielername] 0 Lädt angegebenen Spieler zur Basis ein
    /base kick [Spielername] 0 Der angegebene Spieler wird aus der Basis gekickt
    /base protect 0 Deaktiviert/Aktiviert die Protection der Basis (Kisten, Öfen, Barrels, ...)
    /base close 0 Die Basis wird gelöscht

    Zuletzt bearbeitet am: 1. Dezember 2023 23:03 Uhr

    Profilscreenshots

    Hier ein Guide zum Hochladen von Bildern für die eigene Profilseite

    Befehl Level Beschreibung
    Imgur link erstellen 0 Auf https://imgur.com/ auf "New Post" gehen und danach gewünschtes Bild hochladen
    Link abrufen 0 Über hochgeladenes Bild hovern und auf "Copy Link" drücken
    Zur Profilseite Hinzufügen 0 Kopierten Link als neuen Screenshot hochladen und mit einer Beschreibung versehen
    Warten (: 0 Zuletzt musst du auf die Bestätigung des Bildes durch einen Host warten (ein netter Hinweis kann diesen Vorgang beschleunigen). Natürlich werden nur Bilder gestattet, welche nicht gegen unsere Regeln verstoßen. Nach Bestätigung ist das Bild automatisch auf deinem Profil zu sehen!
